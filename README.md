• funcionalidades implementadas:
    Método para cálculo porcentagens da lista de valores [(valor, percentual)]
    
    Método que efetua o acréscimo de valor conforme a lista enviada [(valor, percentual)]
    
    Método para redução de valor conforme a lista enviada [(valor, percentual)]


• funções criadas:
    calculate_percentage([(valor, percentual)])
    
    add_percentage([(valor, percentual)])
    
    reduce_percentage([(valor, percentual)])


• breve explicação dos cálculos desenvolvidos:
    Para calcular porcentagem:
        valor * valor divido por cem (100)

    Para fazer um aumento de valor:
        valor + (valor * valor divido por cem (100))

    Para fazer o desconto:
        valor +-(valor * valor divido por cem (100))
